Worker Artifact

CS-499-worker is a .NET 8.0 worker service that uses AMQPNetLite to subscribe to multiple AMQ topics from a message broker. It processes the messages, extracts a few entities, and adds them to tables in the "SMAT" database (CS-499-database). It also checks the message barcode in the database and inserts the entities with "New" status if not found, or "Updated" if found.

Data Structures and Algorithms Enhancement

Original Artifact Enhanced Artifact Video Demo
Justification

The legacy system that some of the business logic and database code was pulled from has a history of concurrency issues and dropped messages. For that reason, I wanted to pay extra close attention to how data is being accessed, stored, and updated or deleted. The barcode lookup and lock, in particular, caused issues when multiple messages were received at the same time and processed at the same time on multiple threads or servers. The need to quickly search for the barcode and handle it concurrently made it a good candiate for the Data Structures and Algorithms enhancement.
